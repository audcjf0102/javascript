조건문의 활용

  • if와 else문을 사용, value값을 조건문을 걸어 특정한 명령어를 수행하게 한다.
    ex)
  • input id="red_blue" type="button" value="red" onclick="
    if(document.querySelector('body').style.color === 'red'){ *만약 텍스트 색이 'red'라면
    document.querySelector('body').style.color = 'blue';
    document.querySelector('#red_blue').value = 'red'; *텍스트 색을 'blue'로 바꾸고 id값이 red_blue의 value값을 'red'로 변경한다
    else{ *텍스트 색이 'red'가 아니라면
    document.querySelector('body').style.color = 'red';
    document.querySelector('#red_blue').value = 'blue'; *텍스트 색을 'red'로 바꾸고 id값이 red_blue의 value값을 'blue'로 변경한다
    } }

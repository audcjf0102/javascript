객체

  • 객체 - 함수와 비슷한 개념, 코드의 정리 정돈을 위함, 서로 연관된 함수와 서로 연관된 변수를 같은 이름으로 그룹핑을 하기위한 도구, 디렉토리에 있는 폴더라고 생각. ex)document또한 객체이다.
  • 객체에 속한 함수는 함수라 하지않고 메서드 라고 부른다.
  • 배열은 변수를 순서대로, 0부터 저장시키지만 객체는 이름을 정해 분류할 수 있다.
  • 객체는 var 객체 = {};라는 형식으로 저장된다.
    ex)
    var color = {
    "blue":"pupple",
    "red":"orange"
    };
  • 객체를 사용 할 때는 document.명령어("blue : "+color.blue);
    의 형식으로 사용한다.
  • 객체의 정보를 추가하고 싶을 때는 객체명.객체폴더 = "내용";의 형식이다.
    ex)
    color.white="black";
  • 추가의 다른방법은 객체명["객체폴더"] = "내용";이다.
    ex)
    color["white"] = "black";
    이 방식을 사용하면 객체폴더에 공백을 추가할 수 있다.

객체와 반복문

  • 객체의 반복문은 for를 사용하는데, for(var key in 객체명){};으로 사용한다.
    객체명에 해당하는 객체에 있는 데이터의 key값들의 변수값을 {}에 있는 코드로 설정.
    이 때, key값이란 객체 탭에 나온 '객체폴더'라고 잠시 이름붙인 값을 뜻한다.
    ex)
    var color = {
    "blue" : "pupple"*"blue"가 key값.
    "red" : "orange"*"red"가 key값.
    };
    for(var key in color){
    document.write(key);
    }
    이걸 실행하게 되면 bluered가 출력하게 된다.
  • 또한 document.write(color[key]+' ');처럼 입력을 하게되면
    pupple orange가 출력된다.

객체프로퍼티와 메소드

  • 객체에는 함수도 담을 수 있다.
  • 메소드 - 객체안에 있는 함수.
  • 구성 방식은 객체명.메소드 = function(){}로 구성.
  • 프로퍼티 - 객체에 소속된 변수
  • 객체의 프로퍼티와 프로퍼티를 구분할 때에는 ,가 있어야 한다.

리팩토링(refactoring)

  • 리팩토링 - 코드를 효율적으로 짜 중복된 코드를 낮추고 가독성을 높이는 작업.
    프로그램의 규모가 커질수록 리팩토링을 틈틈히 진행시켜야 한다.
  • this - 이벤트 안에서 일어나는 코드를 가르키는 특수한 키워드.

    ex)input type="button" value="red" onclick="
    if(this.value === 'red'){
    document.querySelector('body').style.color = 'blue';
    this.value = 'red';
    } else{
    document.querySelector('body').style.color = 'red';
    this.value = 'blue';
    }
  • 또한 중복되는 내용을 변수로 지정하여 코드를 더 간략화할 수 있다.
    ex)input type="button" value="red" onclick="
    var as=document.querySelector('body');
    if(this.value === 'red'){
    as.style.color = 'blue';
    this.value = 'red';
    } else{
    as.style.color = 'red';
    this.value = 'blue';
    }

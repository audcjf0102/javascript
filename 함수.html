함수

  • 함수를 사용하는 이유 : 동일한 코드를 사용할 때 중복된 코드를 여러개 쓰는것을 방지함.
  • 함수를 만들 때에는 function 함수명(){명령어}의 형식으로 만든다.
  • 사용 시에는 바디 태그에 원하는 위치에 함수명();을 입력하면 함수가 실행하게 된다.

매개변수와 인자

  • 이 때 변수명 뒤에 오는 즉, ()안에 들어가는 수를 매개변수(parameter)라고 칭한다.
  • 또한 함수를 사용 할 때 쓰는 ()안에 들어가는 수를 인자(argument)라고 칭한다.
  • ex)
    function sum(left, right){*left, right : 매개변수
    document.write(left+right);
    }
    sum(2,3);*2,3 : 인자
    sum(3,5);*3,5 : 인자

리턴

  • 자바스크립트에서는 함수 안에 리턴값을 지정시켜 주어야 한다.
    ex) function sum2(left, right){
    return left+right;
    }

CSS 기초

  • 특정 태그를 디자인 하고싶을때는 css의 style이라는 태그를 사용하고 속성이라는 문법을 사용한다.
  • div 태그 - 어떠한 기능도 없는 명령어를 넣기위한 틀을 잡아주는 태그 단, 줄바꿈이 된다.
  • span 태그 - div와 동일한 태그이지만 줄바꿈이 되지않는다.
  • style 태그 - css문법을 사용한다고 알려주는 태그(j.s의 script와 비슷)
  • span혹은 div로 감싼 class값을 '.클래스값'{명령어}이라는 형식으로 입력하여 같은 class값을 가진 것들에게 같은 효과를 줄 수 있다.
  • 또한 id값을 입력할 수 있는데 이는 class와 비슷하지만 #id값{명령어}라는 형식으로 입력해야한다.
  • class - 그룹핑의 개념.
    id - 개인번호의 개념. : 같은 id값을 지정해 주어서는 안된다.
  • span또는 div도 class와 id 처럼 사용이 가능하다.
  • id선택자의 우선순위가 class선택자의 우선순위보다 높고 class선택자의 우선순위가span, div보다 높다.

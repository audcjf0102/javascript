HTML과 JavaScript의 만남2

  • button을 만드는 방법 : input type="button"이라는 명령어 뒤에 value="a"a라는 자리에 버튼안에 쓰는 문구를 정할 수 있다.
    그 뒤 onclick ="b"b라는 자리에 버튼을 누르면 어떤 효과를 가져오는지 설정할 수 있는 값을 표시 할 수 있다.

    ex) input type="button" value="hi" onclick="alert('bye')"
    를 입력하게 되면 hi라는 버튼이 생기며 버튼을 누르게 되면 bye라는 문구가 창으로 뜨게된다.

    이처럼 웹브라우저 안에서 일어나는 일들을 '이벤트(사건)'라고 한다.

  • 이벤트는 무수히 존재하지만 특수한 이벤트들을 정해놓았다.
    ex)onchange - 입력값이 달라지게되면 명령어를 수행.
    ex)onkeydown - 키를 누르게 되면 명령어를 수행.
